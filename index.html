TypeScript

Into to TypeScript

TypeScript is a strongly types scripting language. It adds additional features and syntax to JavaScript. TypeScript adds static typing to JavaScript which means that developers can add types when using TypeScript. TypeScript is a handy landuage to use when dealing with JavaScript and can help catch errors much earlier then JavaScript can.

Defining Types

TypeScript supports an extension of the JavaScript language, which offers places for you to tell TypeScript what the types should be.

Generics

Generics provide variables to types. A common example is an array. An array without generics could contain anything. An array with generics can describe the values that the array contains..

Classes

TypeScript offers full support for the class keyword introduced in ES2015. As with other JavaScript language features, TypeScript adds type annotations and other syntax to allow you to express relationships between classes and other types.
